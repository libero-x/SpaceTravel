# 🚀 星际航行论坛

一个讨论宇宙航行技术和未来星际旅行路线的在线论坛。

## 功能特点

- 🌌 **宇宙主题设计** - 深空背景，科技感UI
- 📱 **响应式布局** - 适配所有设备
- 💬 **论坛结构** - 推进技术、航行路线、生命支持等板块
- 📊 **实时统计** - 在线用户、话题数量显示
- ✨ **交互体验** - 表单验证、悬停效果

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 渐变, 动画)
- JavaScript (ES6)
- 无外部依赖，单文件运行

## 本地运行

1. 克隆仓库
2. 直接打开 `cosmic_forum.html` 文件

## 在线访问

通过 GitHub Pages 访问：https://[你的用户名].github.io/[仓库名]/
